# E-Commerce Sales Dashboard (Power BI)

## Project Overview
This project presents an interactive Power BI dashboard built to analyze e-commerce sales performance. The dashboard helps businesses understand sales trends, profit distribution, customer purchasing behavior, and payment preferences.

## Objectives
The main goal of this project is to transform raw e-commerce sales data into meaningful business insights using Power BI.

## Key Insights
- Total Sales Amount: 438K
- Total Profit: 37K
- Total Quantity Sold: 5615
- Average Order Value: 121K

## Dashboard Features
- Monthly Profit Trend Analysis
- Profit by Product Category
- Profit by Sub-Category
- State-wise Sales Performance
- Customer Purchase Quantity
- Payment Method Analysis

## Tools Used
- Power BI
- Power Query
- Data Visualization
- Data Cleaning
